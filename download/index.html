System Requirements

  • OS Windows 10 or Windows 11
  • Architecture x64 (64-bit)
  • RAM 4 GB minimum
  • Disk Space 200 MB free
  • IDE Support Any installed IDE (VS Code, JetBrains, etc.)

How to Install

  1. 1
    Click CodeSpace-Setup.exe above and save the installer file.
  2. 2
    Run the installer. Windows may show a SmartScreen prompt — click "More info" then "Run anyway".
  3. 3
    Follow the setup wizard and choose your install location.
  4. 4
    Launch Codespace from the Start Menu or desktop shortcut.
  5. 5
    Add your first project and you're set — no account needed.
